Intelligent Urban Delivery Robot

An AI-based simulation of a delivery robot navigating a 15×15 grid environment using multiple search algorithms.

📌 Features
Grid-based urban environment with:
Obstacles (buildings)
Roads
Traffic zones (higher cost)
Delivery locations
Random generation of 5 delivery points
Visual GUI using Tkinter
Real-time robot movement animation
Performance metrics display (cost, nodes, time)
🧠 Algorithms Implemented
Breadth First Search (BFS)
Depth First Search (DFS)
Uniform Cost Search (UCS)
Greedy Best First Search
A* Search
🔍 Heuristics
Manhattan Distance
Euclidean Distance
⚙️ How It Works
A random grid is generated
Delivery points are assigned
User selects an algorithm and heuristic
Robot computes and follows the optimal path
Process repeats for all deliveries
